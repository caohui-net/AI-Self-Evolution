## 二、目标架构

### 能量循环
```
┌─────────────────────────────────────────────┐
│  其他三个项目（知识生产者）                    │
│  ├─ EvoMap-Integration: 学习 EvoMap 知识     │
│  ├─ bounty-automation: 任务执行经验          │
│  └─ moltbook: 自动化实践                     │
└──────────────┬──────────────────────────────┘
               ↓ 产生知识
┌─────────────────────────────────────────────┐
│  .shared-knowledge（知识池）                  │
│  274 项知识，持续增长                         │
└──────────────┬──────────────────────────────┘
               ↓ 消费知识
┌─────────────────────────────────────────────┐
│  AI-Self-Evolution（知识消费者 + 能力工厂）   │
│  ├─ 读取共享知识                              │
│  ├─ 提炼 Gene/Capsule                        │
│  ├─ 计算 GDI 质量评分                        │
│  └─ 构建记忆图                                │
└──────────────┬──────────────────────────────┘
               ↓ 分发能力
┌─────────────────────────────────────────────┐
│  能力注入（改进执行）                          │
│  ├─ ~/.claude/agents/learned-*.md           │
│  ├─ ~/.claude/skills/learned-*.md           │
│  ├─ {project}/.omc/genes/                   │
│  └─ {project}/CLAUDE.md (追加指令)           │
└──────────────┬──────────────────────────────┘
               ↓ 应用能力
               回到"其他三个项目"，形成闭环
```

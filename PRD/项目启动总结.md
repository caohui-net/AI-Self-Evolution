### 3. 核心知识获取

#### **EvoMap.ai 核心概念**
- **理念**："One agent learns. A million inherit."
- **双协议**：A2A (Agent-to-Agent) + GEP (Genome Evolution)
- **六大资产**：Gene, Capsule, EvolutionEvent, Mutation, ValidationReport, MemoryGraphEvent
- **七阶段生命周期**：Detect → Select → Mutate → Hypothesize → Execute → Evaluate → Solidify

#### **关键技术机制**
- **GDI 评分**：多维度质量控制（结构、语义、信号、策略、验证）
- **记忆图**：因果记忆，经验驱动决策，置信度衰减
- **技能蒸馏**：从成功 Capsule 自动合成新 Gene
- **内容可寻址**：SHA-256 哈希确保完整性
- **.gepx 档案**：主权可移植的进化历史

## 核心发现

### 技术亮点
1. **理论基础扎实**：基于 UC Berkeley 的 Test-Time Training 研究
2. **协议设计完善**：GEP 协议有完整规范和参考实现
3. **安全机制健全**：爆炸半径控制、约束验证、沙盒执行
4. **可移植性强**：.gepx 格式支持跨平台迁移

### 市场机会
1. **需求明确**：AI Agent 开发者需要知识共享机制
2. **竞争适中**：EvoMap 是先行者，但市场空间大
3. **差异化路径**：开源 + 轻量级 + 中文优先

### 技术挑战
1. **质量控制**：需要实现 GDI 多维度评分
2. **知识图谱**：Neo4j + 语义嵌入的复杂性
3. **冷启动**：需要预置基础 Gene 库
